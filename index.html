    Summary: Latitude vs X
    Welcome to my website! The purpose of this beautiful page is to explain a project I once completed that analyzed weather changes. I completed the analysis by pulling data from the OpenWeatherMap API from over 500 cities and using several other programs, such as Matplotlib.

    The main goal for this project is to display the relationship between various factors of weather and latitude. These factors include temperature, cloudiness, wind speed, and humidity. On this website, you may find the four seperate visualizations displaying what I found when looking at these factors as well as their corresponding explanations. You may also find the original data set collected via the API.
